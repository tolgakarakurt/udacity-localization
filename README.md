# udacity-localization
- Kalman Filter (KF)
- Extended Kalman Filter (EFK)
- Monte Carlo Localization (MCL)
### Requirements 
- Visual Studio Code
  - C/C++ Extension
  - C++ Intellisense
  - C/C++ Compile Run

### Installation
- `git clone https://github.com/microsoft/vcpkg` 
- `.\vcpkg\bootstrap-vcpkg.bat`
- `.\vcpkg\vcpkg install eigen3:x64-windows`
